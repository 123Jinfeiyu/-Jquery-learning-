2. let和const声明的变量区别是：

   ```
   let: 普通变量  变量可以先声明后赋值  并且二次赋值 
   const:常量  声明就必须赋值  常量是不能二次修改的 不能改变的
   ```

4. 什么是常量？

   ```
   不变量  且声明就必须赋值  不能重复赋值修改 
   ```
