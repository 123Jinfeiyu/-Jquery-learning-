## 控制流程
1.if判断 
  if(条件){
    // 条件成立 执行里面的代码 
  }

2.switch 判断  
 switch(条件){
    case 值;
    满足上面这个值一个条件 执行语句;
    break; 跳出当前 不执行下一段代码
 }



 3.循环结构  
 >循环是任何语言都会有的一个语法    用来反复执行一段代码  
 for循环   

 写法: for(初始条件; 结束条件;递增){
    循环代码
 }


 4.函数    作用: 封装代码     function
